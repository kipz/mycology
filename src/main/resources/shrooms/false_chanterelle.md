### False Chanterelle
*Hygrophoropsis aurantiaca*

**Quick Reference:**
- **Item ID:** `mycology:false_chanterelle`
- **Edibility:** Inedible (possibly mildly toxic)
- **Toxicity:** Mild
- **Season:** Autumn (August-November)
- **Habitat:** Coniferous woodland

**In-Game Properties:**
- **Nutrition:** 0
- **Saturation:** 0.0
- **Effects:** Weakness I (200 ticks)

**Identification:**
False Chanterelle resembles true chanterelles but has true gills. Cap 3-8cm, funnel-shaped, orange to orange-brown. **Critical difference:** Has true, thin, forked gills (not blunt ridges), orange colored. Stem 3-6cm, orange, often off-center. Grows on conifer wood debris.

**Lookalikes:**
- **Chanterelle** (Cantharellus cibarius) - EDIBLE. Blunt ridges not gills, apricot smell, brighter yellow
